SPARQL Rule Set — Companion Artifact

Abstract

This module is the SPARQL CONSTRUCT rule set implementing the access decision logic for the DPP Access Control framework. The rules operationalise how an AccessRequest is evaluated against applicable AccessPolicy instances to produce an AccessDecision.

SPARQL rules are executable queries, not OWL vocabulary terms, so they are documented here as a companion artifact rather than with the WIDOCO vocabulary template used for the dpp-ac core, roles, data-categories and GDPR modules.

Rule mechanism

The choice of a SPARQL-based rule mechanism (over SWRL or SHACL-AF rules) is recorded in the project decision record docs/decisions/ADR-001-rule-mechanism.md.

The published Turtle source currently carries the module declaration and provenance metadata. The CONSTRUCT rule bodies are maintained alongside the decision record; this page is the stable documentation endpoint for the http://w3id.org/dpp/access-control/sparql-rules IRI and will be updated as rules are published into the graph.
