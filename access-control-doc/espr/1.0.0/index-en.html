Regulatory Policy Set — Companion Artifact

Abstract

This module provides the canonical role–permission matrix for the DPP Access Control framework as normative AccessPolicy instances grounded in ESPR Regulation (EU) 2024/1781, with legal-basis assertions per the GDPR module.

This module is composed of policy instances (ABox), not new vocabulary terms, so it is documented here as a companion artifact rather than with the WIDOCO vocabulary template used for the dpp-ac core, roles, data-categories and GDPR modules.
